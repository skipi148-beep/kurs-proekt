# Отчёт по итогам автоматизации

## Реализовано
Все 12 сценариев из Plan.md полностью автоматизированы с использованием паттерна Page Object Model и выноса данных в класс CardData.

## Трудности и их устранение
Риск использования фиксированных пауз (sleep) был успешно устранен переходом на продвинутый метод динамического ожидания условий СУБД через `WebDriverWait`.
