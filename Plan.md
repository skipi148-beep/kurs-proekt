# План автоматизации тестирования веб-сервиса покупки туров

## 1. Перечень автоматизируемых сценариев

### Позитивные сценарии (СУБД PostgreSQL / MySQL)
1. **Успешная оплата тура по дебетовой карте:** Ввод карты `APPROVED` (4444 4444 4444 4441) с валидным сроком действия и именем владельца -> UI показывает успешное всплывающее окно -> в БД в таблице `payment_entity` создается запись со статусом `APPROVED`.
2. **Успешное оформление кредита по карте:** Ввод карты `APPROVED` через форму кредита -> UI показывает одобрение кредита -> в БД в таблице `credit_request_entity` создается запись со статусом `APPROVED`.

### Негативные сценарии (Дебетовая карта и Кредит)
3. **Отказ в операции по отклоненной карте:** Ввод карты `DECLINED` (4444 4444 4444 4442) -> UI должен показать ошибку -> в БД создается запись со статусом `DECLINED`.
4. **Отправка пустой формы:** Все поля формы остаются пустыми -> под каждым полем отображается сообщение об ошибке валидации, форма не отправляется на сервер.
5. **Невалидный формат номера карты:** Ввод слишком короткого номера (15 цифр) -> под полем "Номер карты" появляется ошибка формата.
6. **Невалидный месяц:** Ввод несуществующего месяца (например, `00` или `13`) -> ошибка "Неверно указан срок действия".
7. **Истекший срок действия (Прошедший месяц/год):** Ввод даты меньше текущей даты -> ошибка "Истёк срок действия карты".
8. **Избыточный срок действия (Карта из далекого будущего):** Ввод года, превышающего текущий более чем на 5 лет -> ошибка "Неверно указан срок действия".
9. **Невалидное поле "Владелец":** Ввод имени кириллицей, цифрами или спецсимволами -> ошибка формата.
10. **Невалидный CVC-код:** Ввод 1 или 2 цифр вместо 3 -> ошибка формата.

## 2. Перечень используемых инструментов
* **Python 3.14 + Pytest** — базовый язык программирования и тестовый фреймворк.
* **Selenium WebDriver** — инструмент управления браузером, реализующий паттерн **Page Object Model**.
* **SQLAlchemy + psycopg2** — библиотеки для отправки прямых проверочных SQL-запросов к СУБД.
* **Allure Report** — система генерации детальных отчетов с шагами тестирования.
